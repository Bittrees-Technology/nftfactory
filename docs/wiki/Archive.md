# Archive

The active documentation surface is the wiki under `docs/wiki`, rooted at [Home](./Home.md).

Older point-in-time documents, PR notes, generated snapshots, and superseded standalone docs are preserved under:

- `docs/archive/`

This includes:

- dated status snapshots
- PR-specific descriptions
- release-note snapshots
- generated contract dependency dumps
- superseded standalone docs that have been absorbed into the wiki

Use the archive for historical reference only. The current source of truth should be the wiki pages linked from [Home](./Home.md).
